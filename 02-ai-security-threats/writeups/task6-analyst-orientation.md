## Objective

Pass an orientation assessment that draws on both the foundations (Room 1) and the threats/defenses (Room 2) to obtain the flag.

## Approach

1. **Consolidate the two rooms.** The gate tests the AI→ML→DL→LLM stack (Room 1) plus the five core vulnerabilities, AI-enhanced attacks, and defensive AI use cases (Room 2).
2. **Answer the scenario prompts.** Map each question to the correct concept — e.g., identify which vulnerability a described scenario represents, or which defensive AI use case applies.
3. **Complete the gate.** Passing yields the flag → `THM{<redacted>}`.

## Concepts tested

- ML lifecycle & model types (Room 1)
- MITRE ATLAS + five AI model vulnerabilities (Room 2, Task 2)
- Deepfake / AI phishing / AI malware recognition (Task 3)
- Defensive AI use cases: analysis, prediction, summarization, investigation (Task 4)
- Baseline controls: RBAC, MFA, ISO/IEC 27090, SHAP/LIME (Task 5)

## Result

Assessment passed; flag captured. Ready for Room 3 (AI Models & Data — supply chain security).
